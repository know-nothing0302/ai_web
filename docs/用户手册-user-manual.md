# AI在徐医 — 使用手册

**版本**: v3.0  
**最后更新**: 2026年6月12日  
**受众**: 徐州医科大学教师、职工、学生

---

## 1. 平台简介

"AI在徐医" 是徐州医科大学内部 AI 资讯平台，面向全校师生提供以下核心服务：

| 功能 | 说明 |
|------|------|
| 📰 **AI 资讯浏览** | 按频道（政策解读、教学应用、科研前沿、工具指南等）浏览 AI 相关文章 |
| 💬 **智能问答 (PageAgent)** | 基于当前页面内容的 AI 助手，支持文章解读、内容分析、关联推荐 |
| 📬 **订阅推送** | 支持邮箱和企业微信推送，可按频道定制，支持每日/每周频率 |
| 🔬 **AI 试验场** | 对比不同 AI 模型回答的实验环境 |

---

## 2. 访问与登录

### 2.1 访问地址

平台部署在内网环境，通过学校统一入口访问。

### 2.2 CAS 单点登录

平台使用学校统一身份认证（CAS）。首次访问时自动跳转登录页：

1. 在浏览器地址栏输入平台 URL
2. 系统自动跳转至 CAS 登录页面
3. 输入您的**统一身份认证账号**（学工号）和密码
4. 登录成功后自动返回平台首页

> **注意**: 登录状态保持 8 小时。过期后需重新登录。

---

## 3. 浏览资讯

### 3.1 首页

首页展示最新发布的 AI 资讯文章列表，支持：

- **频道筛选**: 顶部频道标签可切换不同分类（政策解读、教学应用、科研前沿等）
- **搜索**: 搜索框输入关键词，按标题和摘要匹配
- **加载更多**: 滚动到底部自动加载，或点击"加载更多"按钮

### 3.2 文章详情

点击任意文章卡片进入详情页：

- **文章正文**: 支持富文本排版，包含标题、正文、来源链接
- **AI 摘要**: 文章顶部的 AI 生成摘要，快速了解核心内容
- **原文跳转**: 点击原文链接可查看原始出处

### 3.3 阅读历史

平台自动记录您的阅读历史，侧边栏可查看最近浏览的文章。

---

## 4. 使用智能问答 (PageAgent)

PageAgent 是"AI在徐医"的核心 AI 功能，位于页面底部中央的 **"AI 智能分析与搜索"** 按钮。

### 4.1 打开问答面板

点击底部中央的蓝色按钮，弹出 PageAgent 对话面板。

### 4.2 支持的功能

| 场景 | 使用方法 | 示例 |
|------|----------|------|
| 文章解读 | 在文章详情页打开 Panel，直接提问 | "这篇文章的核心观点是什么？" |
| 内容总结 | 选中一段文字后打开 Panel，AI 自动带入选中内容 | 选中一段复杂的政策文本，问"帮我用白话解释这段话" |
| 站内搜索 | 在任意页面提问相关文章 | "有没有关于 AI 辅助教学的类似文章？" |
| 页面操作指导 | 在管理/订阅页面提问操作相关 | "如何设置每日推送？" |

### 4.3 回答模式

- **精简模式** (默认): 回答控制在 200 字以内，用要点列表，只给核心结论
- **详细模式**: 深度展开，包含背景分析、核心论点、支撑证据、相关案例

点击面板顶部的 "精简" / "详细" 按钮切换。

### 4.4 对话管理

| 操作 | 方法 |
|------|------|
| 新建对话 | 点击 "新建对话" 按钮 |
| 查看历史 | 点击 "历史对话" 按钮，可按页面类型筛选 |
| 搜索历史 | 在历史列表顶部的搜索框输入关键词 |
| 复制回答 | 点击回答下方的 "复制" 按钮 |
| 收藏回答 | 点击回答下方的 ☆ 星标按钮 |
| 分享回答 | 点击分享按钮，复制分享内容到剪贴板 |

### 4.5 使用技巧

- **Shift + Enter** 换行，**Enter** 直接发送
- 选中页面文字后再打开 Panel，AI 会自动理解选中内容
- 在文章详情页提问，AI 会优先依据当前文章内容回答
- 同一会话内 AI 会记住对话上下文

## 5. 订阅推送

### 5.1 订阅设置

访问 **订阅管理** 页面设置推送偏好：

1. 选择订阅的**频道**（可多选）
2. 选择推送**频率**:
   - **每日推送**: 每天晚上 20:00 / 上午 11:00
   - **每周推送**: 每周日晚上 20:00
3. 绑定**企业微信**账号接收推送

### 5.2 今日推送

**今日推送** 页面可查看当天推送的所有文章汇总。

## 6. 个人中心

### 6.1 用户画像

平台会根据您的浏览和提问行为，自动生成个性化画像：

- **兴趣主题**: 您最关注的 AI 领域

对平台功能或 AI 回答有任何意见，可以通过反馈功能提交。您的反馈将帮助改进平台。

### 6.3 排行榜

**排行榜** 页面展示用户的活跃度排名（基于提问数量、文章浏览量等）。

## 7. 常见问题

**Q: PageAgent 能回答哪些问题？**  
A: PageAgent 专为当前页面的内容服务。它能解读文章、总结要点、推荐相关文章、指导页面操作。它不能回答与徐州医科大学无关的通用问题。

**Q: AI 回答不准确怎么办？**  
A: 可以切换"详细模式"获取更全面的分析。如果仍有问题，通过反馈功能提交，我们会持续优化。

**Q: 如何取消订阅？**  
A: 在订阅管理页面，将所有频道取消勾选并保存即可。

**Q: 登录失败怎么办？**  
A: 请确认使用的是学校统一身份认证账号。如果问题持续，联系信息化处。

**Q: 支持哪些浏览器？**  
A: 推荐使用 Chrome、Edge、Firefox 等现代浏览器的最新版本。

**Q: 手机上能用吗？**  
A: 可以。平台支持移动端浏览器访问，PageAgent 面板在手机上也能正常使用。
